# Contributing to Claude Spinner Words

Thank you for your interest in contributing to **Claude Spinner Words**! This project is an open-source technical workstation registry, and we value community-driven enhancements.

## How to Contribute

### 1. Thematic Extensions
We are always looking for new thematic indices. To add a new theme:
- Create a new `.json` file in the `themes/` directory.
- Follow the structure of `official.json` and ensure your verbs are unique.
- Submit a Pull Request with a brief description of your theme's aesthetic.

### 2. Implementation Scripts
If you have improvements for the `apply.ps1` or `apply.sh` scripts (e.g., support for additional OS platforms or configuration types), we welcome your contributions.

### 3. Simulator Enhancements
The `Source Code/` directory contains the technical workstation simulator. If you can improve the hardware-accelerated animations or the diagnostic UI, please open an issue first to discuss your proposal.

## Style Guidelines
- Maintain the **high-fidelity technical aesthetic** of the existing workstation.
- Ensure all JSON files are valid and all verbs follow the canonical 'ing' suffix where appropriate.
- Keep the language in documentation simple and accessible.

## Pull Request Process
1. Fork the repository and create your branch from `main`.
2. Ensure your changes align with the **technical parity** of the Claude Code CLI.
3. Update the `README.md` if your contribution adds a new theme or feature.
4. Submit your PR for review!

We look forward to your contributions and to expanding this registry together!
